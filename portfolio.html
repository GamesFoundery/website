The Imhotep Project is a 3D Environment Art project designed as a graduation artwork. The original idea behind Imhotep was to explore Ancient Egyptian history and their glorious builder's past that allowed the production of a set of buildings amongst the most marvellous architecture wonders of the world. I hope you'll like this project as much as I enjoyed creating it.

MATERIALS

In this gallery you can see my latest PBR (Physically Based Rendering) Materials projects which are created with the Substance Suite and most of all Substance 3D Designer. This software allows users to achieve realistic results. I really felt in love with this software and its way of working with nodes. You'll see several different textures types.
